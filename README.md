# Obstacle Avoiding Robot using Arduino

## Components Used
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- L298N Motor Driver
- Servo Motor
- DC Geared Motors
- 9V Battery

## Working Principle
The robot detects obstacles using an ultrasonic sensor. 
If an object is detected within a threshold distance, 
the robot stops, scans left and right using a servo motor, 
and moves in the direction with more space.
